Grinn AstraADA-1680
===================

The Grinn AstraADA is a core module adapter that seamlessly integrates the Grinn AstraSOM-1680 with the Synaptics Astra Machina Foundation Series demo board. It ensures reliable connectivity, proper routing, and signal integrity for high-speed interfaces, enabling efficient prototyping and validation of embedded solutions. Key features include Ethernet (10/100/1000 Mbps) via Realtek RTL8211F, 128Mb QSPI Flash Memory, HDMI RX input for multimedia processing, and a MicroSD card slot for expanded storage. Its compact and robust design makes it ideal for IoT, industrial automation, and multimedia applications.

The user guide is available in PDF format and can be downloaded from `here <https://grinn-global.com/storage/app/uploads/public/67b/ed9/b92/67bed9b927f14662660535.pdf>`_
