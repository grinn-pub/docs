Grinn GenioEVB-700
==================

Grinn GenioEVB-700 is a versatile evaluation board built to showcase the capabilities of the Grinn GenioSOM-700 module. With a comprehensive set of peripherals, debugging tools, and connectivity options, it accelerates AI, IoT, and multimedia application development. Its well-thought-out design ensures seamless integration of sensors, displays, and communication modules, simplifying prototyping and reducing time-to-market for new products.
The board includes advanced debugging features, Raspberry Pi HAT compatibility, and interfaces for displays, cameras, and wireless communication, making it a perfect reference for developers designing next-generation devices.

The user guide is available in PDF format and can be downloaded from `here <https://grinn-global.com/storage/app/uploads/public/677/d45/230/677d4523050dc432321705.pdf>`_.
